Progressive Web Design

Do you know what Progessive Web Design can offer?

PWD (Progessive Web Design) is the new apps generation! It offers many so many tools that are so close or same to what a native app would offer! Here are some of the main characteristics of a PWA .

  • Push notifications

    sending push notifications to users is one of the most used feature in native apps. This feature is also available in PWD. Push notifcations can ensure user's interactivity with the app is sustained and may lead to increase in sales.(In case the web app in question offers online shopping. )

  • Use Background Sync

    In case of a loss of internet connexion , this feature will allow the user to perform an action (writing a comment , or reply to a message , etc..) that will be saved and applied later as soon as the device estabishes an internet connection.

  • Offline mode

    Offline mode is a very usefull feature that helps a PWD app function fast enough to be mistaken for a native one. (If all other PWD requirements are met or exceeded , eventually.) Using cash memory to store files in the user's browser (images , css styles , external links...) , the app would run smoother and faster as fetching documents won't be for everyfile , since a lot of them are alrady stored within the users's web browser. Oh , also Giving user's some content to interact with in case of bad or non existing internet connexion is better than a "ERR_INTERNET_DISCONNECTED" message display.
